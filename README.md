# Library Management System

## 📌 Description
This project is a Library Management System built using Java Servlets and MySQL. It helps manage books, users, issue and return operations.

## 🛠 Technologies Used
- Java (Servlets)
- HTML
- MySQL
- Apache Tomcat

## ⚙️ Features
- Add Book
- View All Books
- Delete Book
- User Signup & Login

## ▶️ How to Run
1. Install Apache Tomcat
2. Configure MySQL database
3. Update DB credentials in code
4. Run project on server

## 📂 Project Structure
- Java files for backend logic
- HTML files for frontend
